# Architecture Decision Records

Use this folder to capture why you made major technical decisions.

Good ADR topics:
- why AKS and EKS are both implemented
- why Argo CD was chosen for GitOps
- why Key Vault and Secrets Manager are used instead of hardcoded secrets
- why a hub-and-spoke or segmented VPC model was selected
- why Kyverno or Gatekeeper was selected
- why observability tooling was chosen

Each ADR should contain:
- context
- decision
- options considered
- consequences
